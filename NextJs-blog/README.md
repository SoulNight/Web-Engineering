# NextJs-blog

This is a Next.js project with a blog structure.

## Getting Started

First, install the dependencies:

```bash
npm install
```

To run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Dependencies

Below is a list of npm dependencies used in this project:

- `next`: The React framework for production - provides server-side rendering and generating static websites for React based web applications.
- `react`: A JavaScript library for building user interfaces.
- `react-dom`: Serves as the entry point of the DOM-related rendering paths.
- `date-fns`: Modern JavaScript date utility library.
- `gray-matter`: A library to parse front-matter from strings or files.
- `remark`: A library to process markdown content.
- `remark-html`: A plugin to compile markdown to HTML.

Ensure that you have a `.env.local` file with the required environment variables if any are used in the project.

## Scripts

In the project directory, you can run several commands:

- `npm run dev`: Runs the app in development mode.
- `npm run build`: Builds the app for production to the `.next` folder.
- `npm start`: Starts a Next.js production server.

For more information on Next.js, check out the [Next.js documentation](https://nextjs.org/docs).
